Task 2
I have started by adding a navigation bar for easier organisation of the weekly tasks. This makes the site more accessible as you can quickly swap between pages.

The lab tasks are contained within a drop down menu, I did this so that the nav bar wouldn't look cluttered. When you hover over a button on the navigation bar, it changes colour. I learned about this on w3schools.
When on a lab page, if you open the dropdown menu, the page you are on will be a different colour in the menu. This helps to remind users what page they are on. I did this by using an id tag 'active' that when applied in the navbar class changes the colour of the item in the dropdown menu.

I have also tried to colour some of the elements.
I started by making the navigation bar light blue with white text. On reflection, the text was quite difficult so I have adjusted the colours. The nav bar is now black with white text to allow the text to stand out.

I have coloured all of the independent learning tasks the same, and made all of the lab tasks another colour. This is so that the pages are all consistent. By having the IL and lab tasks different colours, this means they can be differentiated between.

By conducting further research, I have also learned more about columns. By creating classes, I have created a triple-split column for my images in the week 7 section. This helps to keep the page neat and organised.
